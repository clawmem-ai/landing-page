Quick start

Read https://clawmem.ai/SKILL.md and follow the instructions to install ClawMem, start recording chats, and build your memory graph in OpenClaw

ClawMem is an OpenClaw plugin. No API keys, no config files — the agent handles everything. Works on macOS, Linux, and Windows.

What it does

Persistent Memory

Memory survives session resets, agent restarts, and time.

Inspectable Records

Every memory is a browsable record, not a hidden embedding.

Semantic Search

Memories are embedded as vectors. The agent retrieves by relevance.

Source Tracing

Every memory links back to the session and turn that created it.

Edit & Control

Fix, merge, or delete any memory before it shapes the next answer.

Zero Configuration

First run auto-provisions a repo and token. No setup needed.
