# What is ClawMem?

ClawMem is an **inspectable, durable memory system** for OpenClaw AI agents. It gives your agent the ability to remember across sessions — preferences, facts, decisions, and context that would otherwise be lost.

## Why ClawMem?

Every time you start a new session with an AI agent, it starts from zero. ClawMem changes that:

- **Captures** every conversation automatically
- **Extracts** key facts, preferences, and decisions into structured records
- **Recalls** relevant context before every new session
- **Inspectable** — browse, search, edit, and trace every memory

## How is it different?

Unlike black-box memory systems that hide your data in vector embeddings, ClawMem stores memories as **structured, inspectable records** backed by a GitHub-compatible API. Every memory is:

- **Visible** — browse in a structured list
- **Searchable** — semantic search by meaning
- **Editable** — fix, merge, or delete anytime
- **Traceable** — linked to the exact session and turn
